# BlockSrvt

## Este projeto foi feito como teste para uma vaga de desenvolvedor front-end para a empresa Block srvt. <br/><br/> A ideia foi criar uma listagem de produtos, aplicando infinite scroll na listagem, os dados vem de uma api publica da empresa.

### 🛠 Tecnologias

As seguintes ferramentas foram utilizadas na construção do projeto:

- [React](https://pt-br.reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Next.JS](https://nextjs.org/)
- [Redux](https://redux.js.org/)
- [Axios](https://axios-http.com/ptbr/)

### 🎲 Rodando o Back End (API)

Você consegue rodar a API do backend entrando na pasta backend e rodando o comando

```bash
yarn node main.js
```

### 🎲 Rodando o Front-End

Você consegue rodar o frontend entrando na pasta frontend e rodando o comando

```bash
yarn dev
```

<h3>Instalar dependencias</h3>

```bash
yarn
```

<h3>Rodando o projeto com somente um comando</h3>
<strong>Você precisa estaa na raiz do projeto e rodar o comando abaixo</strong>

```bash
yarn start-project
```

<p>Com esse comando ele vai navegar entre as pastar, instalar as dependencias necessárias e rodar tanto o branckend quanto o frontend
